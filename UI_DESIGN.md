## Color Scheme

### Main Theme
- Background: Dark Gray (#2E2E2E)
- Text: White (#FFFFFF)
- Accent: Blue (#3498DB)

### Status Colors
- Active/Success: Green (#2ECC71)
- Inactive/Stop: Red (#E74C3C)
- Warning: Orange (#F39C12)
- Info: Blue (#3498DB)

## Button States

### Start/Stop Button
- **Stopped**: Green background, "Start Listening"
- **Listening**: Red background, "Stop Listening"
- **Disabled**: Gray background, dimmed text

### Model Buttons
- **Download**: Blue background, white text
- **Active**: Green border, disabled
- **Inactive**: Gray background, white text

## Responsive Design

The application is designed for:
- **Minimum Resolution**: 800x600
- **Recommended**: 900x600 or higher
- **Scaling**: Kivy handles DPI scaling automatically

## Accessibility

- Large, readable fonts (14-32pt)
- High contrast text
- Clear button labels
- Status indicators with text and color
- Keyboard navigation support (future)
